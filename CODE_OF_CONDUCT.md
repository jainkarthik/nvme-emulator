# Code of Conduct

We are committed to a welcoming, respectful, and harassment-free community.

## Our standards

- Be respectful and constructive.
- Assume good intent and collaborate in good faith.
- Give actionable technical feedback.
- Avoid personal attacks, discrimination, or harassment.

## Scope

This Code of Conduct applies to project spaces, including issues, pull requests,
code reviews, and related community discussions.

## Enforcement

Project maintainers may edit, remove, or reject contributions and comments that
violate this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, report it privately to:

- karthik.jain@tsecond.ai

All reports will be reviewed and handled in good faith.
