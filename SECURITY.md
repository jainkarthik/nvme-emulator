# Security Policy

## Supported versions

This project is currently maintained on the default branch.

## Reporting a vulnerability

Please do **not** open a public issue for security vulnerabilities.

Instead, report privately to:

- karthik.jain@tsecond.ai

Include:

1. A clear description of the issue and impact.
2. Reproduction steps or proof-of-concept.
3. Any suggested remediation.

We will acknowledge receipt and coordinate a fix before public disclosure.
